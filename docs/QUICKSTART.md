# TLC Planner - Quick Start Guide

## Welcome to TLC Planner

TLC Planner is a production-ready calisthenics training application designed to help athletes master bodyweight skills through intelligent progression tracking and calendar-first planning.

## Getting Started

### 1. Sign Up

Visit the homepage and click "Get Started" to create your account. You'll need:
- Email address
- Password (minimum 6 characters)
- Display name (optional)

### 2. Navigate the Dashboard

After signing in, you'll see three main sections:

#### Planner
- Calendar-based workout scheduling
- Track training sessions
- View workout history
- Plan future sessions

#### Exercise Library
- Browse calisthenics exercises
- View muscle groups and progressions
- Search by name or category
- Currently empty - exercises to be added

#### Settings
- Update your profile
- Export your data
- Manage account preferences
- Sign out

### 3. Create Your First Workout

1. Click "Planner" from the dashboard
2. Select a date on the calendar
3. Add exercises and sets (coming soon)
4. Log your workout

## Key Features

- **Neumorphic Design**: Beautiful, modern UI with soft shadows
- **Role-Based Access**: User, Coach, and Admin roles
- **Secure Authentication**: Email/password with optional PIN
- **Data Export**: Download your workout history
- **Responsive**: Works on desktop and mobile

## Technology Stack

- **Frontend**: React 18, TypeScript, Vite
- **Backend**: Lovable Cloud (Supabase)
- **Database**: PostgreSQL with Row Level Security
- **Auth**: Multi-factor with OAuth support
- **Design**: Neumorphic CSS variables

## Environment

This app uses Lovable Cloud for backend services. No additional configuration needed!
